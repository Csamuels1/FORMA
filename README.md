# FORMA

FORMA is a Flutter mobile app for home fitness, nutrition guidance, progress tracking, and accountability.

## Current status

This repository currently contains the foundation docs and Flutter project scaffold. Product implementation will start after the Phase 1-6 documents are reviewed and approved.

## Structure

- `docs/` - source-of-truth product and engineering documents
- `lib/` - Flutter application code
- `assets/` - static app assets

## Local setup

Flutter toolchain is required before running the app.

```bash
flutter pub get
flutter run
```
